
# 👁️ Object Detection System

> Real-time object detection system using pre-trained deep learning models for identifying objects in images/video streams.

---

## 🎯 Problem

Traditional image processing struggles with detecting multiple objects in real-world environments with high accuracy and speed.

---

## 💡 Solution

This project implements an object detection pipeline using deep learning models to:
- Detect multiple objects in images  
- Classify objects with bounding boxes  
- Enable real-time detection  

---

## ⚙️ Features

- 🧠 Detect objects using pre-trained models (YOLO / SSD)  
- 🎯 Draw bounding boxes with labels  
- 📸 Process images and video streams  
- ⚡ Real-time inference capability
